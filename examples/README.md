# Examples

Sample commands extracted by `/amarcord` from real sessions.
These are what gets saved to `~/.config/opencode/commands/` after you run `/amarcord`.

| File | What it captures |
|---|---|
| `mps-torch-dtype-mismatch.md` | PyTorch device mismatch on Apple Silicon MPS |
| `opencode-json-cwd.md` | opencode not inheriting working directory from terminal |

To add your own: run `/amarcord` at the end of a session, it writes directly to
`~/.config/opencode/commands/`. You can copy interesting ones here for reference.
